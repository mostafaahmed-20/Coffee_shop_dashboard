Coffee Shop Dashboard (Power BI)

## 📊 Project Overview
This Power BI dashboard provides insights into the daily operations of a coffee shop, including sales performance, inventory status, Branches, and staff efficiency.

## ✅ Features
- Visual representation of daily, weekly, and monthly sales
- Product-wise performance tracking
- Inventory and stock level monitoring
- Customer feedback analysis
- Employee productivity metrics

## 🛠 Tools Used
- Power BI Desktop
- Excel / CSV (as data source)
- DAX (Data Analysis Expressions)
- Power Query

## 📁 Data Sources
- Sales data
- Products
- branches
- Employee

## 📌 How to Use
1. Open the `.pbix` file in Power BI Desktop.
2. Refresh the data to load the latest updates.
3. Explore different pages/tabs for various insights.
4. Use slicers and filters to customize the views.
